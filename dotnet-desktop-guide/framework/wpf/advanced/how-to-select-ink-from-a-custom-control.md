<span data-ttu-id="99322-103">Aggiungendo un <xref:System.Windows.Ink.IncrementalLassoHitTester> al controllo personalizzato, è possibile abilitare il controllo in modo che un utente possa selezionare l'input penna con uno strumento lazo, in modo analogo al modo in cui il <xref:System.Windows.Controls.InkCanvas> Seleziona l'input penna con un lazo.</span><span class="sxs-lookup"><span data-stu-id="99322-103">By adding an <xref:System.Windows.Ink.IncrementalLassoHitTester> to your custom control, you can enable your control so that a user can select ink with a lasso tool, similar to the way the <xref:System.Windows.Controls.InkCanvas> selects ink with a lasso.</span></span>  
  
 <span data-ttu-id="99322-104">Questo esempio presuppone che si abbia familiarità con la creazione di un controllo personalizzato abilitato per l'input penna.</span><span class="sxs-lookup"><span data-stu-id="99322-104">This example assumes you are familiar with creating an ink-enabled custom control.</span></span>  <span data-ttu-id="99322-105">Per creare un controllo personalizzato che accetti input penna, vedere [creazione di un controllo input penna](creating-an-ink-input-control.md).</span><span class="sxs-lookup"><span data-stu-id="99322-105">To create a custom control that accepts ink input, see [Creating an Ink Input Control](creating-an-ink-input-control.md).</span></span>
