# <a name="flowdocumentscrollviewer"></a>FlowDocumentScrollViewer
Il <xref:System.Windows.Controls.FlowDocumentScrollViewer> controllo viene usato per visualizzare il <xref:System.Windows.Documents.FlowDocument> contenuto in un contenitore di scorrimento. Si differenzia da <xref:System.Windows.Controls.FlowDocumentPageViewer> , che Visualizza il contenuto in base a ogni pagina.
