# <a name="working-with-images-bitmaps-icons-and-metafiles"></a>Utilizzo di immagini, bitmap, icone e metafile
GDI+ fornisce la `Bitmap` classe per l'utilizzo di immagini raster e della `Metafile` classe per l'utilizzo di immagini vettoriali. Le classi `Bitmap` e `Metafile` ereditano entrambe dalla classe `Image`.
